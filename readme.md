# Tree-Visualizer

Welcome to Tree-Visualizer, an interactive web application designed to aid in learning about and visualizing B-trees. This tool is built with React, Tailwind CSS, and Framer Motion.

## Live Demo
Check out the interactive application here: [https://prashanttak04.github.io/B-Tree-Visualizer/](https://prashanttak04.github.io/B-Tree-Visualizer/)

## Features
* **Interactive Sandbox:** Insert, delete, and search keys in real-time.
* **Step-by-Step Animations:** Watch nodes split and rebalance organically.
* **Educational Panels:** Learn the history, properties, and real-world use cases of B-Trees.
